## Menggunakan Lingkungan Virtual Python

Jika Anda lebih suka menggunakan lingkungan virtual Python, langkah pertama adalah menginstal Python di sistem Anda. Kami sarankan mengikuti [panduan ini](https://realpython.com/installing-python/).

Setelah Python terinstal, Anda harus bisa menjalankan perintah Python di terminal. Mulailah dengan menjalankan perintah berikut untuk memastikan instalasi benar sebelum melanjutkan: `python --version`. Ini akan menampilkan versi Python yang tersedia di sistem Anda.

Saat menjalankan perintah Python di terminal (seperti `python --version`), program yang menjalankan perintah Anda adalah Python "utama" di sistem. Kami sarankan menjaga instalasi utama ini bebas dari paket apa pun, dan menggunakannya untuk membuat lingkungan terpisah untuk setiap aplikasiâ€”dengan cara ini, setiap aplikasi memiliki dependensi dan paketnya sendiri tanpa risiko masalah kompatibilitas.

Di Python, ini dilakukan dengan [*lingkungan virtual*](https://docs.python.org/3/tutorial/venv.html), yaitu direktori mandiri yang berisi instalasi Python dengan versi spesifik beserta semua paket yang diperlukan aplikasi. Anda bisa membuatnya dengan berbagai alat, tapi kami akan menggunakan paket resmi Python bernama [`venv`](https://docs.python.org/3/library/venv.html#module-venv).

Pertama, buat direktori untuk aplikasi Andaâ€”misalnya, direktori baru bernama *transformers-course* di root direktori home:

```
mkdir ~/transformers-course
cd ~/transformers-course
```

Dari dalam direktori ini, buat lingkungan virtual menggunakan modul `venv` Python:

```
python -m venv .env
```

Sekarang seharusnya ada direktori bernama *.env* di folder yang sebelumnya kosong:

```
ls -a
```

```out
.      ..    .env
```

Anda bisa masuk/keluar lingkungan virtual dengan skrip `activate` dan `deactivate`:

```
# Aktifkan lingkungan virtual
source .env/bin/activate

# Nonaktifkan lingkungan virtual
deactivate
```

Pastikan lingkungan aktif dengan perintah `which python`: jika mengarah ke lingkungan virtual, Anda berhasil mengaktifkannya!

```
which python
```

```out
/home/<user>/transformers-course/.env/bin/python
```
